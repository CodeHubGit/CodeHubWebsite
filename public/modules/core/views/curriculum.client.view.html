Curriculum

The future is written in full-stack JavaScript, so that’s what we teach at Isomer. We’ll learn on the MeanJS framework (MongoDb, Express, Angular, and Node), which is quickly become an industry standard.

And, concentrating on one language means you’ve got less to learn, which gives you more time to build.

Open Source

All of the code you will write in the class will be Open Source. This means that you can add it to your public portfolio, and showcase your talents to other programmers. Additionally, we will talk about how to utilize and contribute to other open source projects on the web.

Learning to Learn

Learning to code is very different from other skills because of the rapid pace of change in technology. It is very likely that you will be using completely different tools within 2-3 years of learning them. As a result, we emphasize how to develop a curious mind, and engage in live long learning so that you can keep pace with the latest trends.

Startup Skills

Most of the problem solving aspects of creating a start up overlap nicely with the problem solving involved in coding. We'll help you with your pitch, your product direction, and how to create a MVP.

Continued Learning

Once an Isomer, always an Isomer. All students are welcome back at any time. You get lifetime access to instructors’ office hours, to attend all future discussions, and keep training with the class on our current web development projects.

Core Computer Science

No CS degree required. We'll teach you the parts that are actually important from a degree, and we'll skip the parts that aren't.

Data Structures

Data structures are used store non-persistent memory. We cover all of the essential data structures such as arrays, hash maps, lists and trees.

Algorithms & Recursion

An algorithm is a set of rules that are repeated in order to solve a problem. You'll learn how to write them, and how to use some of the more brain-tickling aspects, such as recursion.

Big O & Efficiency

You probably hate it when an app you use is constantly loading. We'll teach you how to analyze run time efficiency to make sure that your code is as efficient as possible.

Abstraction

A computer is built on layers of complex systems. We'll teach you how to navigate these layers to become a computer pro.

Core Software Engineering

Most traditional CS degrees don't cover these topics in depth, but they are essential in the industry.

Object-Oriented & Functional Programming

There are two schools of thought in computer programming. We'll make sure you understand the differences, and when to use them.

Multi-Threaded & Asynchronous Patterns

Most modern computers and mobile devices have multiple processors in them called cores. You will learn how to write code that can run across multiple processors simultaneously.

Debugging

Sometimes fixing problems in your code can be incredibly difficult. We'll show you how to solve these problems using a disciplined approach.

Modularity

Good software can be broken down into individual pieces.

Client-side Libraries

We cover the most important aspects of running code inside of the browser.

Angular & Data Binding

The cutting edge front end frameworks usually include a concept called data binding. Data binding lets you write cleaner, more intuitive code.

The DOM (HTML5 & CSS)

All visual elements on a web page are created using HTML & CSS. Tools such as Bootstrap and Sass will allow you app to be responsive and well designed.

Authentication & Cross Domain

Learn the rules that govern how a browser is allowed to communicate with a server.

Server-side Libraries

We'll make sure you know the ins & outs of running code in the cloud.

Node & NPM

Node allows javascript to run on the server, allow us to build desktop and server applications using the same language we run in the browser.

Express & APIs

Have you ever logged into a website using your facebook credentials? Being able to embed other applications into your own is part of what APIs enable us to do. You will also build your own APIs using a library called express.

Object Relation Mapping

An ORM allows us to simplify how to connect and query a database.

Database Storage

Every app needs a way to save user data. There are many approaches to this problem, and we cover the most relevant strategies.

MongoDB / NoSQL

A new and popular alternative to SQL databases, these can speed up development time and are easy for introducing beginners to pick up.

SQL & Relational Databases

We also cover more complicated SQL databases, which most of the web is built on top of.

DBaaS (Firebase)

A relatively new technology, database as a service abstracts your data store to an easy to use API.

Industry Tools

You'll also be learning useful tools and libraries for JavaScript and web development in general, such as GitHub, Socketio, and Grunt, Bower, & Yo.

Git Version Control

A standard in the industry, we will use git to track our code changes, and manage working on large code bases with other students.

IDEs & Text Editors

There are many different options here, but you will learn how to pick tools that will speed up your development process.

Terminal

Learn how to execute text commands without a GUI.
