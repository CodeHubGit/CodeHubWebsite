Mapping Salt Lake City

Current Class Project

Mapping Salt Lake City is a community-created archive of Salt Lake City’s neighborhoods and people that documents the city’s changes through art, critical and creative literature, personal maps and multi-media projects.

Students are currently updating the existing Mapping SLC website to have more modern user experience, as well as improved functionality for the site's admins.

Technologies Used: Mongo, Express, Angular, Node, Mapbox API, Amazon S3, Location APIs

Fruit For All

Upcoming Class Project

Isomer is currently partnering with Green Urban Lunch Box to develop a mobile and web application. The mobile app will allow volunteers to take photos and track the locations of trees that are ready to have their fruit harvested, which normal an expensive and labor intensive operation.

Technologies Used: Mongo, Express, Angular, Node, Ionic, Hardware GPS, Mapbox API, Amazon S3, Location APIs
